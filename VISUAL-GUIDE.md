## ğŸ¯ Key Concepts

### 1. Separation of Concerns
- **Vercel**: Website hosting (HTML/JS)
- **Cloudflare R2**: File storage (PDFs/videos)
- **Cloudflare Worker**: Upload API
- **Supabase**: Database (metadata)

### 2. URL-Based System
- Files stored in R2
- URLs stored in Supabase
- Library loads URLs from Supabase
- Browser fetches files from R2

### 3. Hierarchical Folders
- `parent_id`: Links to parent
- `depth`: How deep in tree
- `path`: Full path string
- Auto-calculated by triggers

### 4. GitHub Deployment
- Code in repository
- Cloudflare reads from GitHub
- Auto-deploys on push
- No manual editing
